科研展示

目标检测是计算机视觉领域的一个基本问题，在现有的研究中，强监督学习获得了良好的性能，前提是图像标记的高精度性。然而现实生活中对于图像的标记往往耗时耗力且容易受个人主观意识左右。深度学习的日益发展，为降低图像标记的成本带来了可能性。弱监督目标检测也受到越来越多的人关注，目前弱监督目标检测大致分为三类：多实例学习（Multiple Instance Learning），隐变量学习（Latent Variable Learning），深度多实例学习网络（Deep Multiple Instance Learning Network）。本文主要综述了弱监督目标检测的现状和一些新颖的方法，然后对这些方法进行分析比较。弱监督目标检测依然还有很大的发展空间，值得人们对其进行进一步的研究。

--弱监督目标检测

最近，航拍旋转目标检测在计算机视觉中引起了广泛关注。与自然图像中从水平视角拍摄的物体不同，航拍图像通常是从高空鸟瞰拍摄的，这表明航拍图像中的物体通常体积小，方向多样，背景复杂。除此之外，还存在着大长宽比、密集分布和类别极不平衡的挑战。
